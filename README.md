# Instructions
1. Update the WiFi details in Assignment.cpp (lines 10-11) for your network
2. Open the "src" directory in the terminal and run "python app.py" to start the web server
3. Update the server address in Assignment.cpp (line 14) with your server address
4. Flash the ESP32 with the updated program
5. Reboot the ESP32
